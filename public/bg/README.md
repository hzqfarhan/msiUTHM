# Background & Logo Assets

Place your custom images here:

| File | Purpose | Recommended Size |
|------|---------|-----------------|
| `bg-light.jpg` | Light mode background | 1920×1080+ |
| `bg-dark.jpg` | Dark mode background | 1920×1080+ |
| `app-logo.png` | App logo (sidebar + header) | 512×512 PNG transparent |

The app automatically picks `bg-light.jpg` or `bg-dark.jpg` based on the current theme.
Images are rendered at 30% opacity behind the glass layers.
